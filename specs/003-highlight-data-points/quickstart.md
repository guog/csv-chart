# 快速入门：数据点高亮显示

**功能**: 003-highlight-data-points
**前置条件**: 002-csv-chart-visualization 功能已完成

## 测试步骤

### 1. 启动开发服务器

```bash
cd /Users/guogang/GitHub/csv-chart
pnpm run dev
```

### 2. 打开可视化页面

访问 http://localhost:5173/visualization

### 3. 上传测试 CSV

使用 `apps/web/public/sample-data.csv` 或创建测试文件：

```csv
日期,销售额,订单数
2024-01-01,85,50
2024-01-02,120,75
2024-01-03,95,60
2024-01-04,150,100
2024-01-05,80,45
```

### 4. 配置图表

1. X 轴选择：日期
2. Y 轴选择：销售额
3. 图表类型：柱状图

### 5. 测试高亮功能

#### 场景 A: 默认阈值

- 阈值默认为 100
- 预期：销售额 >= 100 的柱子显示红色（120, 150）
- 其他柱子显示蓝色（85, 95, 80）

#### 场景 B: 修改阈值

1. 将阈值改为 90
2. 预期：图表立即更新
3. 红色柱子：95, 120, 150
4. 蓝色柱子：85, 80

#### 场景 C: 禁用高亮

1. 清空阈值输入框
2. 预期：所有柱子恢复蓝色

#### 场景 D: 折线图高亮

1. 切换为折线图
2. 设置阈值为 100
3. 预期：超阈值数据点显示为红色大圆点

#### 场景 E: 多系列高亮

1. Y 轴选择：销售额、订单数
2. 设置阈值为 70
3. 预期：两个系列独立判断高亮

### 6. 验证工具提示

1. 鼠标悬停在红色数据点上
2. 预期：工具提示显示 "⚠️ 超出阈值" 标识

## 常见问题

### Q: 阈值输入框在哪里？

A: 在"配置图表"区域，图表标题输入框下方

### Q: 如何禁用高亮？

A: 清空阈值输入框或删除输入值

### Q: 负数阈值是否有效？

A: 是的，支持负数阈值（如 -50）
