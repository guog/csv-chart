# 快速入门：CSV 图表可视化

## 先决条件

- Node.js 18+
- pnpm

## 设置

1.  安装依赖：
    ```bash
    pnpm install
    ```

2.  启动开发服务器：
    ```bash
    pnpm dev
    ```

## 测试功能

1.  在浏览器中打开 `http://localhost:5173`。
2.  导航到“CSV 可视化”页面（如果已集成到主页，则直接在主页）。
3.  **上传**: 点击“选择文件”并选择一个 CSV 文件（< 10MB）。
    -   *示例 CSV*:
        ```csv
        Date,Sales,Profit
        2023-01-01,100,20
        2023-01-02,150,30
        2023-01-03,120,25
        ```
4.  **预览**: 验证表格是否正确显示数据。
5.  **配置**:
    -   选择 "Date" 作为 X 轴。
    -   选择 "Sales" 和 "Profit" 作为 Y 轴。
    -   选择 "Line Chart"（折线图）。
6.  **可视化**: 验证图表是否渲染。
7.  **交互**: 将鼠标悬停在点上以查看工具提示。

## 故障排除

-   **"File too large"**: 确保您的文件小于 10MB。
-   **图表不显示**: 确保您为 Y 轴选择了数值列。
